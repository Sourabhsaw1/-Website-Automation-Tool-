# 🤖 Website Automation Tool

A Streamlit-based web application that automates website interactions using Selenium — search, navigate, and click links without writing any extra code.

---

## 🚀 Features

- 🔍 Automate search queries on any website
- 🖱️ Click links from search results automatically
- ⚙️ Headless mode — runs without opening a visible browser
- 📋 Step-by-step automation log displayed in real time
- 🎚️ Adjustable wait time between actions
- 📄 Captures final page title after automation

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Core language |
| Streamlit | Web UI |
| Selenium | Browser automation |
| WebDriver Manager | Auto-manages ChromeDriver |

---

## 📦 Installation

```bash
git clone https://github.com/Sourabhsaw1/-Website-Automation-Tool-.git
cd -Website-Automation-Tool-
pip install -r requirements.txt
streamlit run app.py
```

> Make sure Google Chrome is installed on your system.

---

## 🖥️ How to Use

1. Enter the **website URL** (e.g. `https://www.google.com`)
2. Enter your **search query**
3. (Optional) Enter **link text** to click after search
4. Toggle **headless mode** from the sidebar
5. Click **Run Automation** and watch the log
